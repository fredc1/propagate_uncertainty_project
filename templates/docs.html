Rules and Functions for Expression Input

The quick way of conveying the expression input rules is by saying that the expression must follow the compilation rules of python, but with the ^ character instead of ** for exponents. For mathematical expressions this is basically PEMDAS (parens, exponents, multiplication/division, addition/subtraction). Also, you may use only single character lower or upper case variables in your expression. One thing to note is that operators like ^ will consider the next character to be their only operand unless you use parens.

You may also use functions from a specified list that is outlined here. Note that all angles are measured in radians.

  • acos(x) : Return the arc cosine (measured in radians) of x.
  • acosh(x) : Return the inverse hyperbolic cosine of x.
  • asin(x) : Return the arc sine (measured in radians) of x.
  • asinh(x) : Return the inverse hyperbolic sine of x.
  • atan(x) : Return the arc tangent (measured in radians) of x.
  • atanh(x) : Return the inverse hyperbolic tangent of x.
  • ceil(x) : Return the ceiling of x as an Integral. This is the smallest integer >= x.
  • cos(x) : Return the cosine of x (measured in radians).
  • cosh(x) : Return the hyperbolic cosine of x.
  • erf(x) : Error function at x.
  • erfc(x) : Complementary error function at x.
  • exp(x) : Return e raised to the power of x.
  • fabs(x) : Return the absolute value of the float x.
  • factorial(x, /): Find x!. Raise a ValueError if x is negative or non-integral.
  • floor(x) : Return the floor of x as an Integral. This is the largest integer <= x.
  • gamma(x) : Gamma function at x.
  • hypot(x1, ...) : Multidimensional Euclidean distance from the origin to a point. Roughly equivalent to: sqrt(sum(x**2 for x in coordinates))
  • lgamma(x) : Natural logarithm of absolute value of Gamma function at x.
  • log(x, base) : Return the logarithm of x to the given base. If the base not specified, returns the natural logarithm (base e) of x.
  • modf(x) : Return the fractional and integer parts of x. Both results carry the sign of x and are floats.
  • pow(x, y) : Return x**y (x to the power of y).
  • sin(x) : Return the sine of x (measured in radians).
  • sinh(x) : Return the hyperbolic sine of x.
  • sqrt(x) : Return the square root of x.
  • tan(x) : Return the tangent of x (measured in radians).
  • tanh(x) : Return the hyperbolic tangent of x.
  • trunc(x) : Truncates the Real x to the nearest Integral toward 0.
